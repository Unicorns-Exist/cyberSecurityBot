# CyberSecurity chatBot__POE

Cybersecurity Awareness Bot (POE – Part 1)

Project Overview
The Cybersecurity Awareness Bot is a C# console-based application designed to educate users about basic cybersecurity practices. The program interacts with the user through a combination of audio, ASCII art,visual effects and a simple Q&A chatbot system.
This project demonstrates fundamental programming concepts such as object-oriented programming (OOP), file handling, user interaction, and basic UI design in a console environment.

Objectives:
Create an interactive chatbot system
Promote cybersecurity awareness
Apply OOP principles using multiple classes
Enhance user experience using audio and visual elements

 Features include:
1. Voice Greeting, a custom recorded welcome message implemented using System.Media.SoundPlayer
2. ASCII Art Logo, converts an image/emblem into ASCII art
3. User Interaction, includes input validation and personalised responses using the user’s name
4. Chatbot System (Q&A) menu-based interaction
5. Visual Enhancements, console colours for visaul appeal, Styled borders and headers. Chat-style interaction.
   
Program Structure:
The application is divided into multiple classes to ensure clean and modular design:
Program.cs - Controls the flow of the application
voiceGreeting.cs - Handles audio playback
asciiLogo.c - Displays ASCII art and UI elements
botUserActivity.cs - Manages user interaction and chatbot logic

Key Concepts Used:
Object-Oriented Programming (OOP)
Classes and Objects
Automatic Properties
Arrays
Exception Handling
File Handling (audio & image)
